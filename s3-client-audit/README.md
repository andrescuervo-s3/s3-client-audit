# S3 Client Communication Audit

A Claude Cowork plugin that performs comprehensive client communication audits across Slack, Gmail, Google Calendar, Notion, and Google Drive.

## How to Use

Say any of the following:

- "Audit the [client] channel"
- "Run a communication audit on [client]"
- "How's the [client] channel looking?"
- "Pull a report on [client]"
- "Check project health for [client]"

## Required Connectors

| Connector | Used For |
|-----------|----------|
| Slack | Channel history, cross-channel search |
| Gmail | Client correspondence |
| Google Calendar | Meeting events, attendee lists |
| Notion | Meeting notes, action items |
| Google Drive | Document inventory |
